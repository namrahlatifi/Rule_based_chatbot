# Advanced Rule-Based Chatbot

## 🔹 Features
- Regex-based pattern matching
- 20+ FAQs
- Synonym and small talk handling
- Conversation logging
- CLI interface

## 🔧 How to Run
```bash
python run_chatbot.py
```

## 📁 Structure
- `chatbot.py`: Logic
- `utils.py`: Helpers
- `run_chatbot.py`: Interface
- `config.json`: Responses
- `conversation_log.txt`: Logs
